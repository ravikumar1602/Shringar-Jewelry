## SECURITY LAYERS
1. Helmet       вҶ’ HTTP security headers
2. Rate Limiting вҶ’ 100 req/15min (auth: 20/15min)
3. CORS         вҶ’ Whitelist only
4. Mongo Sanitize вҶ’ NoSQL injection prevention
5. XSS Clean    вҶ’ Cross-site scripting prevention
6. HPP          вҶ’ HTTP parameter pollution
7. JWT          вҶ’ Access + Refresh token pair
8. bcrypt       вҶ’ Password hashing (salt: 12)
9. Account Lock вҶ’ 5 failed attempts = 2hr lock
10. Input Validation вҶ’ express-validator on all routes
